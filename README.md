## BeLight
Experiments on graphic programming.

The actual goal is to make a minimal 3D software renderer for learning purposes.


## SDL2

This project requires SDL2 for windowing and other setups. Install using

```sudo apt install libsdl2-dev```

## Build
```
cmake -B build && \
cmake --build build && \
./build/bin/BeLight
```
or

`./build.sh
`
